CMakeEasy

This project is a terminal-based CMakeLists.txt generator. It was designed with the idea of creating these types of files in a much shorter timeframe than physically typing out the full context of a CMakeLists file.

CMakeEasy loosely follows the organizational pattern of a typical CMakeLists file, starting with CMake version requirements and ending with post-compilation commands.

The documentation for this project is formatted as an HTML page, which means it should be easily readable by most web browsers.

Please note that this page will be infrequently updated as needed. It may or may not be very minimalistic, or it may contain image files.
